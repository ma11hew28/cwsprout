Class SC.ScrollView

Extends: SC.View
Defined in: frameworks/sproutcore/views/scroll.js
Version: 1.0

Scroll Views are used throughout SproutCore to provide scrollable areas. Although you can use overflow: auto to provide scrollbar anywhere, using a ScrollView is preferrable because it will also notify child views anytime the view is scrolled.
